An open, searchable database of randomized controlled trials (RCTs) that administered a psychedelic drug — across drugs, conditions, and healthy-volunteer studies. Search and filter the records, inspect the extracted metadata, and export your filtered selection as CSV or RIS to seed your own systematic review.

🚧 Pilot. This is an early preview. Every included study appears below, but full data extraction has only been completed for a subset (look for the Extracted badge). For records still pending extraction, the Drug and Condition facets are auto-derived from the title/abstract and may be imprecise. See About this database for methods and the data dictionary.

About this database

This database is built from our living screening and extraction workflow in Covidence. Records progress through search → title/abstract screening → full-text review → inclusion → data extraction. Every included study is listed here; the extracted metadata is added as each study is processed against our extraction template.

How facets are derived

  • Drug and Condition come from the extraction template when a study has been extracted (Extracted badge). For studies still pending, they are auto-derived from the title/abstract by keyword and flagged auto. Treat auto values as provisional.
  • Outcome domains, arms, comparator, and population are only shown for extracted studies.

Exporting

The CSV and RIS buttons export exactly the rows currently shown (after search and filters). CSV contains the full field set for analysis; RIS imports cleanly into Zotero, EndNote, Mendeley, Covidence, and Rayyan to bootstrap a new review.

Rebuilding

The data on this page is generated from the Covidence exports in analysis/master-db/data/ by analysis/master-db/build_database.py, which writes _data/master_db.json. After dropping in a new export, re-run the script and rebuild the site. See analysis/master-db/CLAUDE.md for the full data dictionary and contributor guide.
